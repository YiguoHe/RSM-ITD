# RSM-ITD
This is the repository of RSM-ITD and RSM-CLIP. 
RSM-ITD is a high-quality remote sensing image-text dataset. It contains a total of 210,515 images and 476,342 image-text pairs.
RSM-CLIP is a powerful remote sensing CLIP model that has achieved state-of-the-art performance on RSITMD, RSICD, and UCM datasets. 
We will be open-sourcing our dataset, model, and code in the upcoming period.
